---

</div>

## Features

- 52-week plan covering the Old Testament
- Links to official Come, Follow Me lessons
- Two reading modes: Lesson Order or Chronological
- Progress tracking with local storage
- Current week highlighting

## Quick Start

```bash
npm install
npm run dev
```
